## Примечания

Лабораторные работы выполнены на **ASP.NET Core Web App (.NET)**, поскольку шаблон **ASP.NET MVC 5** отсутствует в современных версиях Microsoft Visual Studio (2022 и новее).

Разработка велась на **Arch Linux** (I use Arch, btw) с использованием **Neovim**. Конфигурацию редактора можно посмотреть в репозитории [R8Zorg/nvim](https://github.com/R8Zorg/nvim).

Все проекты запускались через CLI:

```bash
dotnet run
```

Используемая версия .NET:

```text
10.0
```

Я не проверял, будут ли проекты корректно запускаться через IDE на Windows, однако при запуске через .NET CLI проблем не возникало.
